Button

Button is an interactive element which is used in websites

Solid Buttons

Buttons come in three different colors. You can access them using btn--primary, btn--secondary, btn--information

Outline Buttons

outline buttons can be accessed using btn--outline

Styled Links

Styled links can be accessed using link class and for color you can use link--primary,link--secondary,link--information

Link Link Link

Icon Buttons

Icon buttons can be accessed using icon--btn class.

Floating Action Button

Floating action button can be accessed using btn--fab class.
