Avatar

Avatars are typically used to display a user profile as a picture. Here we provide 5 different sizes of avatar. You can use them using avatar--xs, avatar--sm, avatar--md, avatar--lg, avatar--xl.

Circle Avatar

For using circular avatar you can use the class avatar--circle.

Square Avatar

For using square avatar you can use the class avatar--square.
